# Regression Operating Layer Frontend

React + Vite + Tailwind frontend for an agentic regression management platform.

## What is included

- End-to-end product journey visualization
- 7 core modules:
  - Test Management
  - Agentic Regression Engine
  - Regression Suite Management
  - Script & Data Repository
  - Execution & Scheduling
  - Reporting & Analytics
  - Platform & Infrastructure
- Mock API-style data for KPI cards and execution table
- Styling aligned to the provided `design-system.mdc` (colors, typography tone, glass-style header, card shadows)

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
